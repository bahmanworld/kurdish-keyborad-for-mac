# kurdish-keyborad-for-mac

How to install:
1. Download `Kurdish.zip` file from this repository
2. Extract `Kurdish` file from ZIP file
3. Now, copy and paste `Kurdish` to `/Library/Keyboard Layouts`
4. Done, that's it
